# Jenifer Teles Portfolio Website

## How to preview
Open `index.html` in a browser.

## Before publishing
1. Replace the placeholder email, LinkedIn and GitHub links on `contact.html`.
2. Add screenshots of your real projects if available.
3. Review all text and change anything that does not reflect your own experience.
4. Test every page on desktop and mobile.
5. Keep a record of the AI prompts and outputs used, as required by your assignment.

## Free publishing options
- GitHub Pages
- Netlify Drop
- Cloudflare Pages

The contact form is a visual demonstration only and does not send data.
